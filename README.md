# Тестовое задание аналитика: продуктовые метрики + A/B-тест

Решение тестового задания на позицию аналитика (оригинальная формулировка —
в `Тестовое задание.docx`).

## Что считается

- **MAU** за ноябрь 2023
- **Среднее DAU** за период
- **Retention 1-го дня** — когортный JOIN по `user_id` между датами
- **Conversion rate** — доля пользователей, увидевших рекламу
- **Среднее число показов** рекламы на пользователя
- **ARPU по группам A/B-теста** (`experiment_group`)
- **Медианный возраст** пользователей через `ROW_NUMBER() OVER (ORDER BY age)`

## Использованные техники
Подзапросы, window functions для медианы, условный подсчёт, GROUP BY.
